Términos y condiciones

Uso de la plataforma Voz Segura

Validación de identidad

Los datos de verificación se usan únicamente para confirmar que el usuario es auténtico y evitar suplantaciones. Esta información no se muestra al personal que revisa denuncias.

  • Se usa solo durante el acceso.
  • No se publica ni se comparte con analistas.

Privacidad del denunciante

El sistema está diseñado para que la identidad del denunciante no sea visible para analistas, operadores ni revisores del caso.

Evidencias

Los archivos se almacenan de forma protegida y se conservan solo el tiempo necesario para gestionar el caso.

  • Evita incluir datos personales innecesarios.
  • Sube solo archivos relacionados con el hecho.

Registros del sistema

Los registros guardan acciones operativas para auditoría y trazabilidad, sin incluir datos personales del usuario.

Uso permitido

No se permite enviar información falsa, malintencionada o contenido que no esté relacionado con la denuncia.

Aceptación

Para utilizar la plataforma, debes aceptar estos términos. Si no estás de acuerdo, puedes volver sin continuar.
